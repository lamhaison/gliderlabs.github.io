Docker Project Kick-start

Starting a project? Get started on the right foot. We can help you set up your projects for proper development workflows, CI pipelines, and eventual deployment pipelines. Learn best practices in making minimal Docker containers and fast build pipelines.


Infrastructure Automation

Docker still requires modern infrastructure automation. We have some of the simplest tools designed specifically for container based architectures, while still giving you solid operational control.


Docker Delivery Pipelines

We've helped design several PaaS projects, but everybody's preferred workflow and deployment requirements are different. Let us set you up with your ideal way to deploy that feels like a PaaS, but does exactly what you want.


Hands-on Consulting via Screenshare

Just want a quick brain dump? Want us to help you optimize your Dockerfile or show you how to use some of our preferred tools? Schedule screen sharing time with us by the hour to get hands-on consulting.


Open-source Bounties

Everything we do is open source. We write it and we use it. If you have features that are missing in a tool we use, let us add it for you. Or if you have an open source tool to build that you just want someone else to take care of, let's talk.


Docker Workshops

We also provide hands-on training on-site from Docker basics to showing you how we'd architect a container-based platform for your application. Just want to get your team on the same page on Docker? We can help you out.
